# Playwright Automation Framework 🚀

This repository contains an automated testing suite built with **Playwright** and **TypeScript**, following the **Page Object Model (POM)** design pattern.

## 🛠️ Features
- **POM Architecture**: Clean and maintainable code.
- **CI/CD**: Integrated with GitHub Actions.
- **Reporting**: Detailed HTML reports with traces.

## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/BILAL-AA/playwright-automation.git`
2. Install dependencies: `npm install.`
3. Run tests: `npx playwright test.`
4. View report: `npx playwright show-report.`
